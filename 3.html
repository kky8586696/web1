JavaScript

When a URL using the javascript: protocol is dereferenced, the user agent must run the following steps: Let the script source be the string obtained using the content retrieval operation defined for javascript: URLs. [JSURL] Use the appropriate step from the following list: If a browsing context is being navigated to a javascript: URL, and the active document of that browsing context has the same origin as the script given by that URL Let address be the address of the active document of the browsing context being navigated. If address is about:blank, and the browsing context being navigated has a creator browsing context, then let address be the address of the creator Document instead. Create a script from the Document node of the active document, using the aforementioned script source, and assuming the scripting language is JavaScript.
